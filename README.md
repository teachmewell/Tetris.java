How to run the Java code locally: 

1. go to powershell (use administrator mode)(if you use windows).
2. download Java locally. It should be "winget install EclipseAdoptium.Temurin.17.JDK"

3. Write Java code.
4.  Navigate to folder where you want to save the file. (write me if you do not know what that means)
5. Save your java code into a file. (for example would your file be Example.java for class Example{})

6. type : "javac name.java" (in our example "javac Example.java")
7. you should now have a new file named Example.class
8. Now write "java Example"

9. and the code should run. If your syntax is invalid, javac will throw an Error.
