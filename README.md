How to run the Java code locally: 

1. go to powershell (use administrator mode)(if you use windows).
2. check if you already have it using "java --version" . If it throws out a version, then skip to step 4. 
3. If you do not have java already, download Java locally. It should be "winget install EclipseAdoptium.Temurin.17.JDK".

4. Write Java code.
5.  Navigate to folder where you want to save the file. (write me if you do not know what that means)
6. Save your java code into a file. (for example would your file be Example.java for class Example{})

7. type : "javac name.java" (in our example "javac Example.java")
8. you should now have a new file named Example.class
9. Now write "java Example"

10. and the code should run. If your syntax is invalid, javac will throw an Error.
